# Members Only

Members only is an exclusive clubhouse where members can write anonymous posts. Inside the clubhouse, members can see who the author of a post is, but outside they can only see the story. The project is built with Express, EJS templates, PostgreSQL, express-validator, Passport and bcryptjs as part of The Odin Project.

## Overview

Users can:
- Create a new account  
- Log in and log out
- Create new messages
- See others messages without user name and posted time
- Join secret club with passcode
- Only club members can see the user names and posted times
- Become club admin using passcode
- Only club admin can delete the messages
- All Information are saved in the database
